# Terminal Style

HCrawler Sentinel includes a deliberately stylized terminal layer inspired by classic green-on-black hacker interfaces.

## Visual elements

- randomized startup banner
- boot-sequence text animation
- matrix-like decorative bursts
- framed mission profile box
- animated phase bars
- colored event markers
- graphical risk bars in the summary

## Plain mode

For CI or machine-friendly output:

```bash
hcrawler crawl https://example.com --audit --no-effects -o report.json -f json
```
