# Security Policy

HCrawler Sentinel is for authorized testing only.

## Allowed

- your own sites
- lab environments
- internal inventory
- authorized security assessment
- CTF targets where crawling is allowed

## Not allowed

- unauthorized crawling
- spam list generation
- credential theft
- stalking or harassment
- bypassing access controls
- abusing third-party systems

## Responsible default

```bash
hcrawler crawl https://example.com --profile safe --audit
```
