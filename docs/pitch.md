# HCrawler Sentinel Pitch

HCrawler Sentinel is an authorized web discovery and lightweight audit CLI.

It combines web inventory, scoped crawling, metadata extraction, technology fingerprinting, weak-signal checks, plugin-based analysis, report generation, baseline comparison, and CI-friendly policy gates.

## Differentiators

- safe-by-default crawling
- audit-style findings
- multi-format reports
- SARIF and JUnit output for CI
- SQLite export for querying
- DOT graph export for link visualization
- baseline comparison
- plugin architecture
- YAML/JSON config
- optional async mode
